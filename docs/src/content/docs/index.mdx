## From zero to live site

```sh
pnpm add -D blogwright

pnpm exec blogwright init       # first-run wizard — writes config/production.jsonc
pnpm exec blogwright bootstrap  # create the infrastructure graph
pnpm exec blogwright deploy     # build in a MicroVM and publish
```

The [quickstart](/getting-started/quickstart/) walks every step, and the
[configuration reference](/reference/configuration/) covers everything beyond
the defaults. Reading by machine? Every page is Markdown — append `.md` to any
URL, or start from [/llms.txt](/llms.txt).
